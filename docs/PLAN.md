# Marknote 개발 플랜

## 기능 추가 예정

### 데이터베이스 (노션 데이터베이스 스타일)
마크다운 문서 내에서 구조화된 데이터를 관리할 수 있는 기능. 노션의 데이터베이스처럼 테이블 형태로 데이터를 입력/조회/관리.

- [ ] 데이터베이스 블록 — 문서 안에 삽입 가능한 구조화 테이블
- [ ] 컬럼 타입 — 텍스트, 숫자, 날짜, 선택(태그), 체크박스, URL 등
- [ ] 정렬 — 컬럼 기준 오름차순/내림차순
- [ ] 필터 — 조건별 행 필터링 (값 일치, 포함, 범위 등)
- [ ] 검색 — 데이터베이스 내 텍스트 검색
- [ ] 저장 형식 — 경량 DB 파일 (SQLite 또는 JSON), 마크다운 문서와 같은 폴더에 `.db` 또는 `.json`으로 저장
- [ ] 뷰 — 테이블 뷰 (기본), 추후 보드/갤러리 뷰 확장 가능
- [ ] 마크다운 호환 — DB 없이 문서를 열면 일반 마크다운 테이블로 fallback 표시

### 개선
- [ ] 탭 전환 시 에디터 undo 히스토리 보존 — EditorState 캐시 접근 실패, 다른 방법 조사 필요
